````mdx
# Função: enviarEmail(para, assunto, corpo, anexos = [])

## Propósito

Envia e-mail transacional via provedor configurado.

## Parâmetros

- `para` _(string | string[])_ — destinatário(s)
- `assunto` _(string)_
- `corpo` _(string | HTML)_
- `anexos` _(array, opcional)_ — `{ filename, content }[]`

## Retorno

- `Promise<{ messageId: string }>` — id do envio

## Exemplo

```ts
await enviarEmail("cliente@email.com", "Pedido confirmado", html);
```

### Observações

- Usa remetente padrão do ambiente (MAIL_FROM).
````
