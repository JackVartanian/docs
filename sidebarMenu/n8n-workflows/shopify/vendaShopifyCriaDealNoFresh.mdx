## Detalhes

- Webhook escuta o evento de venda do Shopify.
- Os dados são mapeados.
- Busca informações do cliente no Fresh.
- Caso exista cadastro no Fresh cria Card no pipeline do Fresh.

## Triggers

- Webhook

## Fluxo

- **Webhook**: Escuta o evento de venda do Shopify
- **Code**: Mapeia os dados do evento
- **HTTP Request**: Busca contato do cliente no Fresh
- **If**: Valida se existe cadastro do cliente no Fresh
- **If (FALSE)**: Encerra o fluxo
- **If (TRUE)**: Continua
- **Code**: Cria objeto para criar deal no pipeline do Fresh
- **HTTP Request**: Cria deal no Fresh no pipeline 'CRM Shopify Ecommerce'
