<Info>Guia para atualizar o certificado SSL do sistema</Info>

A cada 3 meses o certificado expira e um novo deve ser colocado no lugar, essa página mostra o passo a passo para isso

OBS.: Utilize as imagens para melhor entendimento

## Guia

- Quando o certificado estiver perto de ser expirado, um e-mail será enviado avisando para ser renovado.
- O certificado deve ser gerado na plataforma ZeroSSL
  - Os dados de login estão na Planilha
  - Os redirecionamentos já estão feitos, basta gerar o certificado
- Depois de gerado baixe os arquivos
- Acessar a GoDaddy
- Acessar o menu Meu produtos
