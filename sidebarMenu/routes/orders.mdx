# Rota: GET /pedidos

## Descrição

Lista pedidos com filtros opcionais.

## Query Params

- `status` _(opcional)_: `aberto | pago | cancelado`
- `page` _(opcional, default=1)_
- `pageSize` _(opcional, default=50)_

## Resposta (200)

```json
{
  "data": [{ "id": "A1", "total": 1500, "status": "pago" }],
  "page": 1,
  "total": 120
}
```
