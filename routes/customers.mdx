# Rota: POST /clientes

## Descrição

Cria um novo cliente no sistema.

## Requisição

**Body (JSON)**

```json
{
  "nome": "string",
  "email": "string",
  "telefone": "string (opcional)"
}
```

## Respostas

**201 Created**

```json
{
  "id": "123",
  "nome": "Maria Silva",
  "email": "maria@email.com"
}
```

**400 Bad Request**

```json
{ "erro": "Email inválido" }
```

## Regras e Observações

- email deve ser único.
- Validação de telefone somente se enviado.
